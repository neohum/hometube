 # hometube
> 이 저장소는 YouTube 서비스 제작을 위한 연습 프로젝트입니다.
 ## 준비사항

- macOS(>= 11.5), Ubunut(>=20.04)
- Node.js

## HTTP 서버

- Node.js를 사용해서 HTTP 서버를 만들어보자
  - [`Express.js`](https://expressjs.com/)
